# Wine Outlier Comparison

Este projeto contém um notebook para comparar diferentes métodos de detecção de outliers utilizando o dataset Wine da UCI.

## Como usar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Abra o notebook `wine_outlier_comparison.ipynb` em seu ambiente Jupyter ou VS Code.
3. Execute as células em ordem para carregar os dados, treinar os detectores e visualizar os resultados.

## Dependências
- pyod
- scikit-learn
- pandas
- matplotlib

## Sobre
O notebook aplica vários detectores de outliers do PyOD, compara os resultados e faz uma visualização usando PCA.
